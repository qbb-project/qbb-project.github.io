Source code width
=================

The lenght of a row should not exceed the 80-columns limit.

Code indention
==============

In all cases, use spaces instead tabs to indent your code.

Four spaces per level is the preferd indention depth, but you may depart from this rule as long as the depth stays consistent within the file.
