The QBB Project

The QBB project aims at providing a powerful set of facilities to ease the development of state-of-the-art simulations. While the project has a slight focus on quantum mechanics, many of its libraries are nonetheless useful for all kinds of other projects.

Subprojects

Qubus

An autoparallelizing framework for tensor arithmetic supporting distributed and heterogeneous execution environments.

echelon

A HDF5-based library for saving and managing scientific data written in modern C++.
